# Password-Generator

## Description

Week Three's challenge involved adding javascript code to a web application made to generate a random password. 
The app required if statements to allow for user flexibility in selecting what types of characters are randomly selected, as well as the length of the password

## Installation

N/A

## Usage

The user can enter the length of the password, and by typing "y" or "n" they can decide to include uppercase characters, lowercase characters or numbers.
